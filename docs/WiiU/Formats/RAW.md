# RAW files

RAW files (Otherwise known as Loadiine) look like this

```
📁 gamename_folder
┣ 📁 code
┃  ┣ app.xml
┃  ┣ cos.xml
┃  ┣ *.rpx
┃  ┗ *.*
┣ 📁content
┃  ┗ this is the game content, folders will vary 
┗ 📁 meta
   ┣ meta.xml
   ┣ *.jpg
   ┣ bootTvTex.tga
   ┣ bootDrcTex.tga
   ┗ iconTex.tga
```
